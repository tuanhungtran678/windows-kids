# 🌈 Windows Kids

A fun Windows-inspired operating system simulation that runs entirely in the browser!

![Windows Kids](https://img.shields.io/badge/Windows-Kids-blue?style=for-the-badge)
![HTML](https://img.shields.io/badge/HTML5-orange?style=for-the-badge\&logo=html5)
![CSS](https://img.shields.io/badge/CSS3-blue?style=for-the-badge\&logo=css3)
![JavaScript](https://img.shields.io/badge/JavaScript-yellow?style=for-the-badge\&logo=javascript)

---

## ✨ Features

### 🖥 Desktop Environment

* Desktop icons
* Taskbar
* Start Menu
* Custom context menu
* Window management

### 🪟 Windows System

* Open multiple windows
* Minimize windows
* Maximize windows
* Close windows
* Drag windows around

### 🎨 Apps

* Paint
* Calculator
* Games
* Kid Browser
* File Explorer
* Recycle Bin

### 📁 File System

* Virtual file system
* Create folders
* Explorer navigation
* Desktop folders

### 🌈 Extras

* Boot Screen
* Wallpaper changer
* Live clock
* Kid-friendly design

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/yourusername/windows-kids.git
```

2. Open the project folder

```bash
cd windows-kids
```

3. Launch with Live Server

```text
Right Click → Open with Live Server
```

4. Enjoy Windows Kids! 🎉
